# OpenCode 102 — Automatización y personalización

Curso avanzado para aprender a automatizar tareas y personalizar
[OpenCode](https://opencode.ai) en tu trabajo diario con
documentación.

## ¿Para quién es este curso?

Para **docentes, redactores técnicos y cualquier persona** que ya
use OpenCode y quiera ir más allá: crear agentes a medida, skills
reutilizables, versionar documentos con Git, conectar sistemas
externos y colaborar en equipo.

Se recomienda haber completado OpenCode 101 o tener soltura con
el manejo básico de OpenCode.

## Cómo usar este curso

1. Abre `index.html` en tu navegador
2. Navega al paso que te interese desde la tabla o el índice lateral
3. Cada paso tiene ejercicios prácticos ("Manos a la obra")
4. Ten OpenCode abierto en la misma carpeta mientras lees

## Requisitos

- OpenCode instalado
- Soltura básica con OpenCode (modos Plan/Build, referencias @)
- Un navegador web para ver los HTML
- Quarto (opcional, solo si quieres modificar las fuentes y regenerar)

## Regenerar los HTML

```bash
bash render-all.sh
```
