# OpenCode 102 — Automatización y personalización

Curso de **7 pasos** para ir más allá del uso básico de
[OpenCode](https://opencode.ai): crear agentes a medida, skills
reutilizables, versionar documentos con Git, conectar sistemas
externos mediante MCP, automatizar tareas con CLI, gestionar
permisos y colaborar en equipo.

## ¿Para quién es este curso?

Para **docentes, redactores técnicos y profesionales** que ya
usan OpenCode y quieren automatizar su flujo de trabajo. Se
recomienda haber completado OpenCode 101 o tener soltura con
los modos Plan/Build y referencias `@`.

> **Curso anterior:** [OpenCode 101 — Curso básico para creadores de contenido](https://github.com/palazon/ocCursoIntro)  
> 12 pasos para aprender los fundamentos: sistema de archivos, prompts, contenido externo, investigación, traducción, tablas, comandos y revisión.

## Cómo empezar

### A. Online (sin instalar nada)

Abre <https://palazon.github.io/ocCursoAvanzado/> en tu navegador.

### B. Desde la distribución local (recomendado)

Abre `cursoAvanzadoOpencode/index.html` en tu navegador. Todo está
listo para seguir los ejercicios con OpenCode abierto a su lado.

### C. Desde los fuentes

Los archivos `.qmd` (formato Quarto) están en `contenidos/`. Para
regenerar los HTML:

```bash
bash render-all.sh
```

## Requisitos

- OpenCode instalado
- Soltura básica con OpenCode (modos Plan/Build, referencias `@`)
- Navegador web para ver los HTML
- Quarto (opcional, solo para modificar fuentes)
